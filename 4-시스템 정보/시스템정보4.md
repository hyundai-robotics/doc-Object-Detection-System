# Object Detection System

4.4	센서 유닛

4.4.1	기능

 센서 유닛은 아래와 같은 기능을 수행한다. 
-	설정된 감지 범위 내 객체의 움직임을 감지
-	감지된 객체의 움직임에 대한 신호를 제어 유닛으로 전달
-	자가 진단을 통하여 결함을 감지하고, 결함이 감지된 경우 해당 정보를 제어 유닛으로 전달

4.4.2	성능

 센서 유닛의 주요 성능은 아래와 같다.

| 성능             | 최소   | 최대   |
|------------------|--------|--------|
| 감지 거리 (mm)   | 500    | 5000   |
| 감지 속도 (m/s)  | 0.1    | 5.0    |
| 수평 감지 각도(°) | ±5     | ±55    |
