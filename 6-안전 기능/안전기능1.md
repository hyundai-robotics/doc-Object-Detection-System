# Object Detection System

## 6.1	안전 기능 개요

시스템은 아래와 같은 안전 기능을 수행한다.
-	접근 감지: 감지 영역 내 객체가 감지되는 경우 시스템을 안전 상태로 전환한다.
-	재시동 방지: 로봇의 예기치 않은 재시동을 방지하여 안전한 상태를 유지한다.

각 안전 기능은 IEC61508: 2010의 SIL2와 ISO13849-1: 2023의 PL d 수준을 만족한다. 두 가지 안전 기능 모두 로봇을 직접적으로 제어하지 않는다. 시스템에서 출력된 유도 신호를 로봇의 안전 관련 메인 제어 시스템이 수신하여 로봇을 제어하고, 결과에 대한 피드백을 시스템으로 다시 전달한다. 시스템은 수신한 피드백을 통하여 안전 기능이 올바르게 작동하였는지 확인할 수 있다. 만약 출력한 유도 신호와 피드백 결과가 일치하지 않을 경우 안전 상태로 전환한다.
